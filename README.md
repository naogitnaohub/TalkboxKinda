# MATLAB Dual-Band LPC Talkbox Effect

A digital simulation of a classic talkbox audio effect implemented in MATLAB using Dual-Band Linear Predictive Coding (LPC). 

This project takes an instrument signal (carrier) and shapes its spectrum using the formant characteristics extracted from a vocal signal (modulator).

## How It Works
1. **Dual-Band LPC Analysis:** The vocal signal is split into low and high frequency bands, then broken into short, overlapping frames. Separate LPC coefficients are calculated for each band.
2. **Residual Extraction:** The spectral envelope is isolated from the voice signal.
3. **LPC Synthesis:** The time-varying vocal filter applies the modulator's spectral envelope directly to the carrier instrument.

## Prerequisites
* MATLAB
* Signal Processing Toolbox

## File Structure
* `talkbox.m` - The main script to configure parameters, load audio, and run the dual-band LPC talkbox effect.
* `talkbox_report.m` - A brief report of the project.
* `/SAMPLES` - Contains the input audio files (vocal modulators and instrument carriers) used for testing.
* `/PLOTS` - Stores the visual charts generated by the report script (LPC envelopes, frequency responses. time-doamin signal).
* `/OUTPUTS` - The destination directory where the talkbox output audio files are saved.

## How to Use
1. Clone this repository or download the ZIP file.
2. Place your carrier and modulator audio files in the `/SAMPLES` folder.
3. Open `talkbox.m` in MATLAB.
4. Update the file paths to point to your audio files.
5. Run the script to generate and play the processed talkbox audio.

## Parameters to Adjust
You can fine-tune the effect inside the script by modifying the configuration structure (`cfg`) and processing variables:

### Windowing
* `cfg.frame_len_ms`: Window length in milliseconds (default: `20`)
* `cfg.overlap`: Window overlap ratio (default: `0.5` for a 50% overlap)

### LPC Order (Dual-Band Configuration)
* `cfg.P_low`: Modulator filter order for low frequencies (default: `48`).
* `cfg.P_high`: Modulator filter order for high frequencies (default: `44`). 
* `cfg.P_x`: Carrier filter order (default: `48`). 

### Stereo Effects
* `stereo_delay_ms`: Stereo delay time in milliseconds (default: `26`).
